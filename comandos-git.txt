Lista de Comandos Git Aprendidos:
git init - Inicializa um repositório Git
git clone - Clona um repositório remotogit add - Adiciona arquivos à área de staging
git commit - Confirma as alterações no repositório localgit status - Mostra o status dos arquivos
git branch - Lista branches ou cria uma nova branchgit checkout - Muda para outra branch
git push origin <branch> - Envia branch para o repositório remoto
